Overview

The project is an IoT-based smart monitoring and irrigation device for potted plants. It continuously measures:

  • Soil moisture → detects when watering is needed
  • Ambient temperature & humidity → ensures optimal growth
  • Air quality (VOC/CO₂) → checks proper circulation
  • Light intensity → verifies plant receives enough light

The system can automatically water the plant using a pump when the soil becomes too dry, while also allowing remote monitoring and manual control via Wi-Fi and cloud integration.

Objectives

  • Monitor soil moisture, temperature, humidity, light, and optionally air quality.
  • Automate irrigation when soil drops below threshold with safety limits.
  • Remote access: cloud dashboards, graphs, and manual irrigation commands.
  • Reliability & safety: fail-safes, OTA updates, and optional water level sensor.

Technologies

  • ESP32 DevKit v1 (Wi-Fi microcontroller)
  • Capacitive Soil Moisture Sensor
  • BH1750 light sensor
  • BME280/BME680 for temp, humidity, VOC
  • Peristaltic pump + MOSFET driver
  • 12V power supply + buck converter
  • Software: Arduino IDE/PlatformIO, MQTT, Cloud dashboards (Adafruit IO, Blynk, ThingsBoard)

Future Plans

  • Expand system for larger gardens with multiple zones.
  • Centralized dashboard for all irrigation zones.
  • Integrate weather forecasts for smarter watering.
  • Explore solar-powered sustainable setups.
